## Problema que resuelve

Muchas personas tienen su computadora lenta y no saben qué hacer sin miedo a romper algo.

Esta herramienta guía paso a paso para mejorar el rendimiento sin conocimientos técnicos.

## Público objetivo

Personas sin conocimientos de informática que quieren intentar resolver problemas básicos antes de pagar soporte técnico.

## Objetivo

- Ayudar al usuario
- Generar confianza
- Ofrecer soporte cuando el problema es más avanzado
